The ATM Playground is a touchless, gesture-driven web app built with MediaPipe Hand Landmarker and React. This guide gets you from zero to a running local instance as fast as possible.

<Note>
  A functioning webcam is required. The app uses your camera feed for hand detection and gesture recognition — no webcam, no gestures.
</Note>

## Zero-install option

If you just want to try the demo without any local setup, open the CodeSandbox template and the app spins up instantly in the browser preview:

**[Open in CodeSandbox →](https://codesandbox.io/p/sandbox/quizzical-neco-svhglk)**

You can fork the sandbox (`Ctrl+S` / `Cmd+S`) to save a copy and edit the code. Skip to the [Using the app](#using-the-app) section if you go this route.

## Local setup

<Steps>
  <Step title="Prerequisites">
    Make sure you have the following installed:

    - **Node.js** v18 or later — [download here](https://nodejs.org/en/download)
    - **npm** (bundled with Node.js)
    - A modern browser with webcam access (Chrome recommended)

    Verify your Node.js version:

    ```bash
    node --version
    ```
  </Step>

  <Step title="Clone the repository">
    Clone the `googlesamples/mediapipe` repository:

    ```bash
    git clone https://github.com/googlesamples/mediapipe.git
    ```
  </Step>

  <Step title="Navigate to the project">
    The ATM Playground lives inside the `tutorials/atm_playground` subdirectory:

    ```bash
    cd mediapipe/tutorials/atm_playground
    ```
  </Step>

  <Step title="Install dependencies">
    Install all npm packages (including `@mediapipe/tasks-vision`, React, Redux, and Workbox):

    ```bash
    npm install
    ```

    This may take a minute on a fresh clone.
  </Step>

  <Step title="Start the development server">
    Start the Create React App development server:

    ```bash
    npm start
    ```

    The server compiles the app and watches for file changes.
  </Step>

  <Step title="Open the app in your browser">
    The app opens automatically at:

    ```
    http://localhost:3000
    ```

    If it does not open automatically, navigate there manually. When prompted, **allow** webcam access. The Hand Landmarker model (`hand_landmarker.task`) loads from the `public/` directory — wait a moment for initialization to complete before gesturing.
  </Step>
</Steps>

<Tip>
  For the best experience, use the app in a **well-lit environment** on a **larger screen**. Good lighting dramatically improves hand detection accuracy. Position your hand clearly in the camera frame before gesturing.
</Tip>

## Using the app

Once the app loads and detects your hand, a cursor appears at your index finger tip. The four core gestures are:

| Gesture | Hand shape | Action |
|---------|------------|--------|
| **HOVER** | Index finger extended | Move cursor over elements |
| **CLICK** | Thumb and index finger pinch | Select / activate element |
| **BACKSPACE** | Open palm, right hand (all fingers extended + wrist below middle DIP) | Undo last input (same as pressing Backspace key) |
| **GRAB** | Closed fist | Open info modal |

See [Gesture Recognition](/guide/gesture-recognition) for the full detection logic.
