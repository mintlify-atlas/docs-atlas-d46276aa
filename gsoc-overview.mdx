## What is Google Summer of Code?

[Google Summer of Code](https://summerofcode.withgoogle.com/) (GSoC) is a global program that offers contributors an opportunity to work on open source projects under the guidance of a mentor from a participating organization. Contributors receive a stipend and collaborate with experienced developers to deliver meaningful improvements to real-world projects over the course of a summer.

## Project Summary

The project aimed to build an interactive web demo showcasing the capabilities of the MediaPipe [Hand Landmarker](https://developers.google.com/mediapipe/api/solutions/js/tasks-vision.handlandmarker) task. The result is the **ATM Playground** — a touchless ATM simulation where users interact entirely through hand gestures detected from a webcam feed.

Google's [MediaPipe Solutions](https://developers.google.com/mediapipe/solutions/guide) helps developers add machine learning to end-user devices including mobile, web, and IoT. The Hand Landmarker task accurately detects and tracks **21 hand landmarks** per frame, which this project uses to power a fully gesture-driven UI.

## Key Contributions

- Developed a fully functional touchless ATM demo using the MediaPipe Tasks Vision API
- Implemented custom gesture recognition logic without using the `GestureRecognizer` task — built directly from raw hand landmark data
- Added offline support with Workbox service workers and `stale-while-revalidate` caching
- Deployed on [Netlify](https://atm-playground.netlify.app) and exported as a [CodeSandbox template](https://codesandbox.io/p/sandbox/quizzical-neco-svhglk)
- Documented the project with a comprehensive tutorial and four blog posts

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="What is Google Summer of Code?">
    GSoC is an annual program by Google that connects open source organizations with contributors. Accepted contributors work on a defined project over the summer under the guidance of an experienced mentor from the organization.
  </Accordion>
  <Accordion title="Which organization was this project under?">
    This project was completed under the **TensorFlow** organization. The work was contributed to the [googlesamples/mediapipe](https://github.com/googlesamples/mediapipe) repository.
  </Accordion>
  <Accordion title="What is MediaPipe Hand Landmarker?">
    MediaPipe Hand Landmarker is a task from MediaPipe Solutions that detects and tracks **21 key points** on the human hand from an image or video stream. It supports both `IMAGE` and `VIDEO` running modes and can detect multiple hands simultaneously.
  </Accordion>
  <Accordion title="Why was an ATM chosen as the demo subject?">
    ATMs are a high-traffic shared touchscreen device with clear hygiene concerns, making them an ideal candidate to demonstrate the practical value of touchless gesture-based interaction. The use case is immediately relatable and showcases the technology's real-world applicability.
  </Accordion>
  <Accordion title="Where can I read the blog posts from the GSoC journey?">
    All blog posts are published at [blog.neilblaze.live](https://blog.neilblaze.live). They cover topics including the community bonding period, a holistic preview of MediaPipe, building the interactive web demo, and predicting custom gestures.
  </Accordion>
