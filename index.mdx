## Welcome

This is the journey book for Pratyay Banerjee's **Google Summer of Code 2023** project with [TensorFlow](https://www.tensorflow.org), building an **interactive touchless ATM demo** powered by MediaPipe's Hand Landmarker.

The project enables users to interact with an ATM interface entirely through **hand gestures** — no physical touch required. Four gestures (HOVER, CLICK, UNDO, GRAB) control a full ATM transaction flow, all computed client-side in the browser using the `@mediapipe/tasks-vision` library.
