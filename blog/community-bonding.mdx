## About the Organization

TensorFlow is an open-source machine learning framework developed by Google that has gained widespread adoption among researchers, developers, and practitioners worldwide.

**MediaPipe**, within the TensorFlow ecosystem, is an open-source framework built on TensorFlow that enables the development of cross-platform pipelines for various media processing applications. It focuses specifically on:

- Video analysis and gesture recognition
- Augmented reality effects
- Real-time perception pipelines
- Cross-platform deployment (mobile, desktop, web)

Being part of the TensorFlow organization means working alongside a vibrant and diverse community with a strong commitment to open-source collaboration, innovation, and knowledge sharing.

## Initial Project Planning

The project centers on building **Interactive Web Demos using the MediaPipe Machine Learning Library**, specifically a touchless ATM playground that demonstrates contactless gesture-based interaction.

Key decisions made during the bonding period:

- **Technology stack**: React.js + Redux + Tailwind CSS for the frontend
- **ML library**: MediaPipe `@mediapipe/tasks-vision` for hand tracking and gesture recognition
- **Core feature**: Use the Hand Landmarker task's 21 landmark points to drive a cursor on screen
- **Custom gestures**: Train a custom gesture recognizer using MediaPipe Model Maker
- **Offline support**: Implement a Workbox-based service worker for offline caching

<Note>
  The final ATM Playground demo is live at [https://atm-playground.netlify.app](https://atm-playground.netlify.app). It requires webcam access and works best in well-lit environments on larger screens.
</Note>
